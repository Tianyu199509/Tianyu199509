## 正在做的项目

### DeskBox

一个轻量级 Windows 桌面整理工具。

它可以帮你：

- 收纳桌面文件
- 映射已有文件夹
- 记录待办事项
- 保存随记和最近复制内容
- 在桌面上控制音乐

我希望它尽量保持 Windows 原生质感，少引入不必要的复杂东西，做得克制一点、稳定一点、好用一点。
